💡 Ví Dụ Sử Dụng API
Các ví dụ sử dụng curl để gọi API từ dòng lệnh

1️⃣ Truy Vấn Dữ Liệu Cảm Biến (Sensor History)

Lấy dữ liệu tilt của tất cả các device, tối đa 10 bản ghi:

curl -s "http://localhost:5000/api/sensor-history?sensor_type=tilt&limit=10"

Lấy dữ liệu vibration của DEVICE001 từ 01/04/2026 đến 08/04/2026:

curl -s "http://localhost:5000/api/sensor-history?sensor_type=vibration&device_id=DEVICE001&start_date=2026-04-01T00:00:00&end_date=2026-04-08T23:59:59"

2️⃣ Xuất Dữ Liệu CSV

Xuất tất cả dữ liệu displacement thành file CSV:

curl -s "http://localhost:5000/api/export/csv?sensor_type=displacement" > displacement_data.csv

Xuất dữ liệu rainfall của DEVICE002 với tên file custom:

curl -s "http://localhost:5000/api/export/csv?sensor_type=rainfall&device_id=DEVICE002&filename=mua_thang04" > mua_thang04.csv

3️⃣ Xuất Dữ Liệu JSON

Xuất tất cả dữ liệu temperature thành file JSON:

curl -s "http://localhost:5000/api/export/json?sensor_type=temperature" > temperature_data.json

Xuất dữ liệu tilt từ 05/04 đến 08/04 và định dạng JSON đẹp:

curl -s "http://localhost:5000/api/export/json?sensor_type=tilt&start_date=2026-04-05T00:00:00&end_date=2026-04-08T23:59:59" | python3 -m json.tool

4️⃣ Kết Hợp với Các Tool Khác

Xuất CSV và mở với Excel/LibreOffice:

curl -s "http://localhost:5000/api/export/csv?sensor_type=vibration" > vibration.csv && libreoffice vibration.csv

Truy vấn dữ liệu và lọc chỉ các thông tin cụ thể:

curl -s "http://localhost:5000/api/sensor-history?sensor_type=temperature&limit=100" | python3 -m json.tool | head -50

📋 Tham Số API

Tham Số Kiểu Mô Tả
sensor_type String Bắt buộc. Loại cảm biến: tilt, vibration, displacement, rainfall, temperature, gnss
device_id String Tùy chọn. Lọc theo device cụ thể (ví dụ: DEVICE001)
start_date ISO DateTime Tùy chọn. Ngày/giờ bắt đầu (ví dụ: 2026-04-01T00:00:00)
end_date ISO DateTime Tùy chọn. Ngày/giờ kết thúc (ví dụ: 2026-04-08T23:59:59)
limit Integer Tùy chọn. Số bản ghi trả về (mặc định: 50, tối đa: 1000)
offset Integer Tùy chọn. Bỏ qua N bản ghi đầu (dùng cho phân trang)
filename String Tùy chọn. Tên file khi xuất (mặc định: sensor_data)
